# AIR Discovery - Product Overview

AIR Discovery is a travel booking platform that combines AI-powered personalized recommendations with flight booking capabilities. The system uses conversational AI to understand traveler preferences and generate customized travel itineraries.

## Core Features

- **AI Chatbot**: Interactive chat interface that conducts interviews to identify user travel preferences, interests, and budget
- **Personalized Recommendations**: Generates up to 3 destination recommendations with detailed day-by-day itineraries based on user profile
- **Flight Booking**: Integration with Amadeus API for real-time flight search and booking
- **Payment Processing**: Stripe integration for secure payment handling
- **Wishlist Management**: Users can save travel plans (itinerary + flights) for future reference
- **Authentication**: AWS Cognito-based user authentication with social login support (Google OAuth)

## Target Users

Travelers seeking personalized vacation planning assistance with integrated flight booking capabilities.

## Business Model

Travel booking platform with AI-enhanced discovery and recommendation engine.
